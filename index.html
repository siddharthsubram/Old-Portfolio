My Interests

The Web is my canvas,

I am interested in Data visualization, interaction design, and UX Engineering The web is my canvas, and I want to create front ends and designs that captivate users and tell a story.

I believe that strong designs are pivotal for any project, as regardless of how breakthrough a new technologies may be, if users cannot intuitively interact with it they will never be able to experience or appreciate th value that the new technology. Good design not only allow the users to easily interact with technology, but also enhances their experience overall. Therefore, the question I'd like to tackle headfirst is as follows: How can I build intuitive and eloquent designs that maximize user experience and impact?

I am currently an aspiring graduate student, and I want to go to graduate programs in the field of human-computer interaction, design, or visualization-focused data science to gain the skills and practical knowledge on how to build designs that allow for the best possible experiences for users while providing them with impactful insights that tell a story, and bring a bit more enjoyment to their days.

Multi-point triangulator

Data Visualization

Interface to view a triangulated structure of a set of defined points.

  • P5
  • Computational Geometry
  • Geometry
Trangulates a set of points based on user input. User can input and clear canvas with left and right mouse buttons respectively. Also displays centroids and circumcircles. Used matrix determinants in order to handle edge collision of triangle sides.

Able to accurately track the edge and triangle quantities, even in scenarios of breaking existing triangles. Started on implimentation of Delauney triangulation.
